## JavaFX Wordle game

A simple Wordle game built with JavaFX, where player gusses a 5-letter word.
The game pulls words from an XML file and highlights letters. 

# Features

  - 5-letter word input
  - Changes to green when a letter is in the correct place
  - Changes to yellow when a letter exists in the word
  - Random words from an XML file (more to be added)
  - Pop-up "Win" or "Lose" alert
  - "Invalid Word" message
  - Restart button

# Technologies

  - Java
  - JavaFX
  - XML
